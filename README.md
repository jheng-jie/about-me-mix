# About Me Mix

多框架 `Static Site Generation` 專案測試

該專案使用 `Yarn Workspace` 用於管理依賴，共用部分靜態資源或程式碼

主要用於技術研究、面試展示，並非實際應用。

> [Next 14](./projects/react/README.md)
